## Course Overview
This class covers advanced topics in Kubernetes, including:
- Helm (package management)
- Multus (multi-network support)
- Telemetry (monitoring and observability)
- Istio (service mesh)

You will use **k0s** as the Kubernetes distribution, which is lightweight and easy to run on various Linux platforms and WSL2.

## Prerequisites
- Basic knowledge of Kubernetes
- Familiarity with Linux command line
- Access to a Linux system (Debian, SUSE, Red Hat, or WSL2)
